## Getting Started

1. make sure you have [`nodejs`](https://nodejs.org/en/) installed

2. clone the repo

```bash
git clone https://github.com/kjscedev/Abhiyantriki_2021.git
cd Abhiyantriki_2021
```

2. In your terminal:

```bash
npm run pull
```
